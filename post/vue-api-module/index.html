Vue - Api接口模块化

Posted on 10-31 2019，2 min read

文件介绍

了解并新建以下目录文件

@apis api管理文件夹

@apis/index.js api接口的通一出口

@apis/base.js 接口域名管理

@apis/xxx.js 更多接口模块

@apis/filing.js 示例接口模块 用于定义filing模块接口列表

index

/**
* @author City
* @description api接口的通一出口
*/

//  导出接口
//  归档模块接口
import filing from '@/apis/filing'



//  导出接口
export default{
    filing
}

base

/**
 * @author City
 * @description 接口域名管理
 */


const base = {
    anLink: '/api'
}


 export default base;
